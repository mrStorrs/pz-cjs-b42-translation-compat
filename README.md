# CJS B42 Translation Compat

Adds B42 JSON translation keys for active legacy translation files.

## Mod Info

- Mod ID: `cjsB42TranslationCompat`
- Author: CJ Storrs

## Layout

- `42/` contains the Build 42 mod files loaded by Project Zomboid.
- `Common/` or `common/` contains shared assets when this mod uses them.

## Notes

This repository tracks the Project Zomboid mod source used by the local development workspace.
